Webhook Admin — Zapier Configuration

Paste Zapier's webhook URL below to receive real-time event notifications. Events automatically send to Zapier when statuses change.

Paste your Zapier webhook URL from "Catch Raw Hook" in your Zap.

Zapier Setup Instructions

  1. Log in to Zapier → Create new Zap
  2. Choose "Webhooks by Zapier" as trigger
  3. Select "Catch Raw Hook"
  4. Copy the webhook URL that Zapier generates
  5. Paste it above and click "Save"
  6. Configure your action (email, Slack, etc.)
  7. Click "Test" button above to verify connection
  8. Activate your Zap

Events That Automatically Send to Zapier

  • ✓ Event submitted (new or resubmitted)
  • ✓ Event approved by VP
  • ✓ Event denied by VP
  • ✓ Event marked as executing
  • ✓ Event completed with report

Common Zapier Actions

  • Email: Send notification to stakeholders
  • Slack: Post message to #events channel
  • Google Sheets: Append event to spreadsheet
  • SMS: Send text alert to coordinator
  • Microsoft Teams: Post to team channel
